Автоматизация логирования активности и настройка сети ВМ

Данный проект содержит набор PowerShell-скриптов для автоматического сбора данных о времени и пользователях в системе Windows 10, а также руководство по конфигурации сетевого интерфейса виртуальной машины.
📁 Структура проекта

    C:\Scripts\Logger.ps1 — основной скрипт, выполняющий запись данных.

    C:\Scripts\CreateTask.ps1 — установщик, который регистрирует задачу в Планировщике Windows.

    C:\Scripts\log.txt — файл лога (создается автоматически после первого запуска).

🚀 Инструкция по развертыванию
Шаг 1: Подготовка папки

Создайте папку C:\Scripts и поместите в неё файлы Logger.ps1 и CreateTask.ps1.
Шаг 2: Создание задачи

Запустите терминал PowerShell от имени администратора и выполните скрипт установки:
PowerShell

Set-ExecutionPolicy RemoteSigned -Scope Process
C:\Scripts\CreateTask.ps1

Шаг 3: Проверка

Через 1-2 минуты проверьте наличие файла log.txt. Каждая запись в нем будет выглядеть следующим образом:
2026-04-18 12:00:00 - Username
🌐 Настройка сети (Bridge Mode)

Для того чтобы виртуальная машина была видна в локальной сети и получила адрес от DHCP-сервера (роутера), следуйте инструкции:

    В гипервизоре (VirtualBox/VMware):

        Перейдите в настройки сети ВМ.

        Выберите тип подключения: Сетевой мост (Bridged Adapter).

        Выберите активный сетевой адаптер хоста (тот, через который идет интернет).

    Внутри виртуальной машины:

        Откройте PowerShell и обновите сетевой адрес:
        PowerShell

        ipconfig /release
        ipconfig /renew

        Проверьте полученный IP: ipconfig. Он должен быть в той же подсети, что и хост-система.

🛠 Технические подробности скриптов
Скрипт логирования (Logger.ps1)

Использует командлет Get-Date для получения точного времени и переменную $env:USERNAME для идентификации текущей сессии. Данные записываются в кодировке UTF8 для поддержки кириллицы.
Планировщик задач

Задача регистрируется под именем "UserActivityLogger".

    Контекст выполнения: SYSTEM (работает в фоновом режиме без открытия окон).

    Интервал: 1 минута.

    Условия: Разрешен запуск при питании от батареи (актуально для ноутбуков).

📄 Удаление

Чтобы удалить задачу из системы, выполните:
PowerShell

Unregister-ScheduledTask -TaskName "UserActivityLogger" -Confirm:$false
